Un array \(b = [b_1, b_2, \dots, b_n]\) tal que \(b_1 = a_n\), \(b_2 = a_{n-1}\), \dots, \(b_n = a_1\).

    \item \textbf{Entrada:} \([1, 2, 3, 4, 5]\) \\
    \textbf{Salida:} \([5, 4, 3, 2, 1]\)
    \item \textbf{Entrada:} \([7, 3, 8, 9]\) \\
    \textbf{Salida:} \([9, 8, 3, 7]\)
    \item \textbf{Entrada:} \([12]\) \\
    \textbf{Salida:} \([12]\)
\end{itemize}
